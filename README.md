# iseeya
A simple video conferencing app that uses [PeerJS](https://peerjs.com/) and [WebRTC](https://webrtc.org/)

## Setup env variables
On root create `.env` file with the following content...

```
# development
HOST=localhost
PEERJS_PORT=3000
PORT=3000

# deploy
HOST=domain.com
PEERJS_PORT=443
PORT=3000
```
